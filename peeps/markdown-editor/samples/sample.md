# Welcome to OpenPeep

OpenPeep is a **modular file viewer and editor** that uses plugins called *Peeps* to handle different file types.

## Features

- **Workspaces** — organize your files into named spaces
- **Peeps** — plugin-based viewers and editors for any file type
- **PeepHub** — discover and install community-built peeps
- **Themes** — macOS, Windows, and Linux native styling

## Getting Started

1. Add a workspace folder in Settings
2. Browse your files in the sidebar
3. Click any file to open it with the best matching peep
