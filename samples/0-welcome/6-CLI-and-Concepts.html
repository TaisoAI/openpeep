Command Line

Manage OpenPeep

$ npx openpeep # Start (wizard on first run)
$ npx openpeep stop # Stop the server
$ npx openpeep status # Check if running
$ npx openpeep restart # Restart
$ npx openpeep doctor # Diagnose issues
Key Concepts

How it all works

🧩

Peeps

Self-contained HTML apps that render specific file types. Each peep is a folder with an index.html and peep.json manifest.

📁

Spaces

Workspaces pointing to your project folders. Add as many as you want. Each space has its own status columns.

🔍

Matching

When you open a file, OpenPeep checks all peeps and picks the best one by extension, content, or folder pattern.

📦

Three-tier System

Peeps load from three places: workspace (project-specific), user (~/.openpeep/peeps/), and built-in.

🎨

Themes

Adapts to your OS. macOS gets Liquid Glass, Windows gets Fluent, Linux gets GTK.

⚡

Live Reload

Edit files externally and OpenPeep updates the preview instantly. Great for dev workflows.

Happy peeping.
