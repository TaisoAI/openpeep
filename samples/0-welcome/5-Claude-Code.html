Claude Code + OpenPeep

Create content with AI, preview it instantly

OpenPeep includes an MCP server that connects to Claude Code. Claude can see your files, know what peeps are available, and create content that renders beautifully.

Two commands, that's it

Run these inside Claude Code to install the OpenPeep plugin. It sets up the MCP server, skills, and agents automatically:

# Run these inside Claude Code
claude> /plugin marketplace add TaisoAI/openpeep
✓ Marketplace taiso-openpeep added

claude> /plugin install openpeep@taiso-openpeep
✓ MCP server: openpeep
✓ Skills: create-content, peep-workflow
✓ Agent: openpeep-creator

Don't have Claude Code yet?

Install it from claude.ai/code, then run the commands above.

1

Install the plugin

Inside Claude Code, run /plugin marketplace add TaisoAI/openpeep then /plugin install openpeep@taiso-openpeep. This installs the MCP server, skills, and agents. You only need to do this once.

2

Start Claude Code in your Space's root folder

Each Space in OpenPeep has one or more root folders. cd into one and run claude. Files Claude creates there will show up in OpenPeep's sidebar. Hit the refresh button to see new files.

3

Ask Claude to create files

Claude creates the file, you click it in OpenPeep, and it renders with the matching peep.
Ask Claude → file created → see it live.
