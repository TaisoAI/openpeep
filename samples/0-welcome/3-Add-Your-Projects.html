Getting Started

Add your own projects

OpenPeep started you with a demo workspace. Here's how to add your real project folders.

1

Open Settings

Click the gear icon (⚙️) in the bottom-left of the sidebar.

2

Add a Space

Under Spaces, click Add Space. Give it a name and add one or more root folders — these are the directories OpenPeep will browse.

3

Browse your files

Your files appear in the sidebar. Click any file and OpenPeep picks the best peep to render it. Images get a viewer, JSON gets an editor, HTML gets a preview — every file type gets its own app.

Example: adding a project
📁 My App ← add this as a root folder
 ├── README.md → Markdown editor
 ├── package.json → JSON editor
 ├── architecture.mmd → Mermaid diagram
 ├── logo.svg → SVG viewer
 └── standup.mn → Meeting notes
