# Welcome to OpenPeep!

Every file type deserves its own app. OpenPeep matches files to custom viewers and editors called **Peeps**.

## Explore the Demo

This workspace has four levels of sample files. Click any file to see its Peep in action:

### 1-Basics
Standard files you already know — text, images, SVG. OpenPeep gives them real viewers.

### 2-Structured
Markdown, JSON, CSV — files you know, but with rich editors (live preview, syntax highlighting, sortable tables).

### 3-Custom Types
Meeting notes, slide decks, and mermaid diagrams. DSL files that map to purpose-built apps.

### 4-Bundles
Entire folders become apps — project boards, dashboards, and more.
